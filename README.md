# AI生成AI
由AI生成AI0.1版本更新至现在的2.6版本
目前支持导入框架修改编程，导入空白文件夹制作以及AI一键生成框架
基础功能：控制台，日志，权重设计，训练，AI聊天等
开发者：玄曦雪
0.1版本是由Deepseek一键生成AI但由于不完善失败
1.0版本是使用模型开源框架为地基Deepseek为教师AI制作的
2.0版本-2.5是完善和修复了一写列的BG并且接入了文件夹导入和文件导入
2.6版本接入了框架生成和框架检测机制
